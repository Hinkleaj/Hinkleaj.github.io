TseNC-Pro CNC Machine

What

  • Commissioned redesign of an open source CNC machine to solve the problems of the original
  • Automate chip removal in the form of a flood coolant system and enclosure
  • The new machine should be able to cut metal all day long without supervision

    • How

  • Designed using Solidworks
  • Coolant system pushes chips to end of the bed where the coolant is filtered and recirculated
  • Hanging the gantry below the cutting bed prevents chip buildup on the linear rails
  • Created a comprehensive BOM including Make-or-Buy decisions and vendor sourcing for COTs parts

    • Result

  • Created engineering drawings for over 100 parts
  • Created an assembly manual with each step illustrated using exploded CAD assemblies
  • Solved the problems of the original, improved the functionality of the original, and gave the improved version back to the open source community

    • IVP Solver

    What

  • Co-project with Sophia D√∂ring aimed at implementing three different numerical methods for solving IVPs
  • Comparing efficiency and accuracy of solvers using different types of test problems with known analytical solutions
  • Implement project in MATLAB

    • How

  • Methods: constant step size implicit Euler solver using Newton-Raphson method, adaptive step algorithm based on ratio between first and second derivatives, and adaptive step algorithm based on minimizing MSE
  • Test problems: Linear differential equation, a nonlinear differential equation, and an ODE with variable coefficients

    • Result

  • Demonstrated advantages of using adaptive step size algorithms for numerical simulations, particularly for solving stiff differential equations or problems with rapidly varying solutions
  • Code is annotated and presented in detail, in a clear and easy-to-follow format
  • Externally relevant results and discussion

    • ROS Robot

    What

  • Created and configured a ROS workspace to implement publisher/subscriber nodes and robot control scripts

  • Simulated motion in Gazebo, RVIZ to visualize sensor data, and scripted actions in Python

    • Result

    • Lego NXT Segway Robot

    What

  • Program a Lego NXT Segway using MATLAB Simulink to implement a controller in the model
  • Simulate both linear and nonlinear systems and compare them using reasonable initial conditions
  • Create a state space controller to control the robot

    • How

  • Set up a non-linear model of the plant based on differential equations, linearized the system, and generated State Space Model from state equations
  • Checked for controllability using pole placement method to obtain the controller gains for the state space controller

    • Result

  • Generated a closed loop control system for the state space model and plotted results for comparison
  • Implemented State Space controller and applied it to the non-linear model
  • Achieved stable control of physical Segway Robot
